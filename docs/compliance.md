# Compliance Notes

Black Ink Signal is intentionally scoped to public, legally accessible information.

## Allowed
- Publicly accessible Reddit content
- Public web pages that do not require authentication
- Rate-limited compliant fetching
- Local-only storage and manual review workflows

## Not allowed
- Bypassing authentication or paywalls
- Scraping private groups, private Discord servers, or gated communities
- Credential theft or session hijacking
- Mass unsolicited messaging automation
- Hidden browser evasion specifically intended to defeat access controls

## MVP source policy
MVP v1 ships with a Reddit public connector stub only.
Future connectors should be added only after source-specific compliance review.
