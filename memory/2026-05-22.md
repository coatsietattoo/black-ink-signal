- Tones: soft/helpful, direct consultation, funny/casual
- Context-aware: different scripts for memorial, coverup, first tattoo, sleeve, artist search
- Copy-to-clipboard buttons in the drawer UI

- GET /stats/revenue — revenue summary from booked leads
- Value map: small=$150, half_day=$500, full_day=$1000, sleeve_project=$3000
